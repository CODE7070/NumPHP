## Installation

### Composer

Simply add a dependency on `numphp/numphp` to your project's `composer.json` file if you use [Composer](http://getcomposer.org/) to manage the dependencies of your project.

    {
        "require": {
            "numphp/numphp": "dev-master"
        }
    }
